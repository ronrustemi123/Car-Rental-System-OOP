# Car Rental System — C++

A simple car rental management system built with C++ structures for the OOP course at SEEU.

## Features

- Add cars and customers
- Rent out a car to a customer
- Return a car
- View all cars, customers, and active rentals
- Find the most rented car

## How to Use

The program runs in a loop showing a menu. Type the number and press Enter.

**Renting a car (option 3)**
1. Pick a customer from the list
2. Pick an available car from the list
3. Enter start day and end day (1–30)
4. The total cost is calculated automatically

**Returning a car (option 4)**
1. Pick the rental from the list
2. The car becomes available again

The program comes with 3 cars and 2 customers preloaded so you can test it immediately.
